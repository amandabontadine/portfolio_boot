Meus Projetos

  • API de Gerenciamento de Tarefas

    Desenvolvi uma API RESTful para um sistema de gerenciamento de tarefas, projetada para ajudar equipes a organizar e acompanhar suas atividades diárias. A API foi construída com Node.js e Express, utilizando um banco de dados MongoDB para armazenar informações sobre tarefas, projetos e usuários.

  • API de Reserva de Hotéis

    Desenvolvi uma API para um sistema de reserva de hotéis, com o objetivo de fornecer uma plataforma robusta e escalável para gerenciar reservas de quartos, disponibilidade e pagamentos. A API foi implementada em Python usando o framework Django e um banco de dados PostgreSQL para garantir integridade e escalabilidade dos dados.
